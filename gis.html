WebApps
  • Old Bucharest
  • An application that reveals the areas of high risk buildings in case of an earthquake of medium / high intensity.
  • The application is developed using the technical expertise to assess buildings suitability.
  • Official data can be found on: mdrap.ro and pmb.ro
  • Food&Drink Turnu-Măgurele
  • The main locations where you can relax when you are visiting Turnu Magurele.
  • In a pleasant ambiance you can find a wide range of Romanian and International dishes.
  • The information was colected in March 2016, from the field.
  • 24/7
  • Do you want to find a 24/7 place in Bucharest? Now you can.
  • The application shows all the non stops in Bucharest and it’s a Leaflet product.
  • The application is developed using all the non stop data from websites. Last update was in February 2016.
  • Flood Areas in Turnu Magurele
  • An application that reveals areas of high flood risk.
  • The application is developed using digital elevation model and flow rivers.
  • It`s more like a scenario, to see what will happen if there is an increase with 10cm in the rivers.

DataBase
  • Street
  • This data show the outline of all streets located in Turnu-Măgurele, Romania.(2016)
  • The shapefile format spatially describes geometries as polyline.
  • Each geometry has a set of associated attributes. Broadly speaking these are a bit like osm's tags.
  • EPSG = 3844
  • Building
  • This data show the outline of all buildings located in Turnu-Măgurele, Romania.(2009)
  • The shapefile format spatially describes geometries as polygons.
  • Each geometry has a set of associated attributes. Broadly speaking these are a bit like osm's tags.
  • EPSG = 3844
  • Trees
  • This data show the footprint of all trees located in Turnu-Măgurele, Romania.(2016)
  • The shapefile format spatially describes geometries as points.
  • Each geometry has a set of associated attributes.
  • EPSG = 3844
